# Service Entry Example

Languages: English | [中文](README.zh-CN.md)

This bundle is a minimal engine-level contract example for a reusable service
entry. It demonstrates:

- project manifest structure
- service entry file references
- input and output schema files
- a declared interval trigger
- artifact output from service code

Validate it with:

```bash
agent-compose bundle validate ./examples/service-entry
```
